💡 ความรู้ที่ควรทราบ:
   🔐 Token Scopes และสิทธิ์การเข้าถึง:
   ├─ ❌ read:public_repo → อ่านได้อย่างเดียว (ไม่สามารถสร้าง Release)
   ├─ ❌ read:repo → อ่านได้อย่างเดียว (ไม่เพียงพอ)
   └─ ✅ repo → ควบคุมได้ทั้งหมด (สร้าง Release ได้)

   🏛 Repository Types:
   ├─ 🌐 Public Repository: ต้องมี "repo" scope
   └─ 🔒 Private Repository: ต้องมี "repo" scope (บังคับ)

   ⚡ Performance Tips:
   ├─ Token มีอายุ 90 วัน (แนะนำ)
   ├─ เก็บ Token ใน .env file เพื่อความปลอดภัย
   └─ หลีกเลี่ยงการแชร์ Token กับผู้อื่น
